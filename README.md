# 🎵 Zen Music Widget : Moteur de Recherche Musical Interactif

[![License: MIT](https://img.shields.io/badge/License-MIT-purple.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/Version-1.2.0-blueviolet.svg)](#)

Ajoutez une bibliothèque de musiques zen et relaxantes à votre site web en moins de 30 secondes. Ce widget interactif permet à vos visiteurs de rechercher et d'écouter des fréquences d'apaisement sans jamais quitter votre plateforme, boostant ainsi votre rétention et votre SEO.

---

## 🚀 Pourquoi intégrer ce widget ?

L'intégration de contenu multimédia interactif est un levier majeur pour être reconnu comme une autorité par les moteurs de recherche et les IA de recommandation.

* **Rétention Maximale (Dwell Time) :** En offrant une ambiance sonore à vos lecteurs, vous augmentez le temps passé sur vos pages, un signal positif crucial pour le référencement.
* **Expérience Utilisateur (UX) Fluide :** Contrairement aux intégrations classiques, ce widget permet de naviguer dans une bibliothèque complète. La vidéo se lit directement dans l'interface pour ne pas rompre la navigation.
* **Respect de la Vie Privée (RGPD) :** Le lecteur utilise le mode `youtube-nocookie.com`, évitant les traçages publicitaires intrusifs.
* **Design Moderne :** Interface épurée "Glassmorphism" compatible avec tous les styles de sites (méditation, yoga, blogs, plateformes d'apprentissage).

---

## 🛠️ Installation Rapide

Copiez et collez ce bloc de code dans votre page HTML, votre barre latérale (sidebar) ou à la fin d'un article :

```html
<div style="width: 100%; max-width: 350px; height: 500px; margin: auto;">
    <iframe 
        src="[https://zen-music.intuitionlibre.com/widget.html](https://zen-music.intuitionlibre.com/widget.html)" 
        width="100%" 
        height="100%" 
        frameborder="0" 
        style="border-radius: 20px; overflow: hidden; border: none;"
        title="Moteur de recherche Musique Zen"
    ></iframe>
</div>
